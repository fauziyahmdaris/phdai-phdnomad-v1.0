# Project Summary
DrPhDAI is an AI-powered platform designed to assist PhD researchers in navigating their academic journey. It enables users to connect their personal GAI accounts (ChatGPT, Claude, Gemini, Grok, Qwen, and Manus) to access specialized tools for research proposal structuring, literature reviews, methodology design, thesis writing, and general academic support. The platform has been enhanced to include a comprehensive tool called "PhD Research Proposal" that guides users through the proposal development process with advanced prompt templates.

# Project Module Description
The platform consists of several functional modules:
- **GAI Connection Hub**: Interface for connecting personal GAI accounts.
- **PhD Research Proposal**: A step-by-step tool for developing research proposals, including keyword generation, gap analysis, title creation, and aims development.
- **Research Navigator**: Assistant for conducting literature reviews and synthesizing findings.
- **Literature Matrix**: Comparative analysis tool for organizing research studies.
- **Thesis Weaver**: Guidance for writing thesis chapters and methodologies.
- **AI Companion**: Support for brainstorming, problem-solving, and motivation.

# Usage
1. **Install Dependencies**: Run `pnpm install` to install all project dependencies.
2. **Build the Project**: Use the command `pnpm run build` to create a production build.
3. **Run the Application**: Use `pnpm run lint` to check for linting issues and ensure code quality.
